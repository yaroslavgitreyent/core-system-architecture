# Розділ 4. Функціональні сценарії користувачів (User Stories)

Для деталізації вимог до системи "EcoSmart City" було розроблено наступні сценарії використання:

## 4.1. Сценарії для мешканців міста
* **US1:** Як мешканець міста, я хочу бачити карту забруднення в реальному часі, щоб обрати безпечний маршрут для прогулянки з дитиною.
* **US2:** Як користувач, я хочу отримувати push-сповіщення на смартфон, якщо рівень токсичних речовин у моєму районі перевищує норму, щоб вчасно зачинити вікна.

## 4.2. Сценарії для екологів та аналітиків
* **US3:** Як еколог, я хочу завантажувати статистичні звіти за місяць у форматі PDF, щоб надати їх міській адміністрації для планування екологічних заходів.
* **US4:** Як фахівець, я хочу порівнювати дані з різних датчиків за певний період, щоб виявити джерело постійного забруднення.

## 4.3. Сценарії для адміністраторів
* **US5:** Як адміністратор, я хочу мати панель керування станом датчиків, щоб швидко ідентифікувати пристрої, які вийшли з ладу або потребують калібрування.
