Практична робота №1
Тема: Створення GitHub-репозиторію та налаштування SSH-доступу

Мета роботи:
Ознайомитися з системою контролю версій Git, створити власний репозиторій на GitHub та налаштувати SSH-підключення без використання логіну та паролю.

Хід роботи:

1. Створено обліковий запис на платформі GitHub.
2. Створено публічний репозиторій first-practice.
3. Встановлено систему контролю версій Git.
4. Встановлено текстовий редактор Visual Studio Code.
5. Згенеровано SSH-ключ за допомогою командного рядка.
6. Додано публічний SSH-ключ до налаштувань GitHub.
7. Перевірено успішне підключення через SSH (без введення логіну та паролю).
8. Репозиторій клоновано на локальний комп’ютер через SSH.
9. Створено файл роботи та виконано commit і push до віддаленого репозиторію.
